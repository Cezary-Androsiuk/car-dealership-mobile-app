# Car Dealership Mobile App

## About

University project - a mobile application written in C++ with the Qt framework, featuring a QML-based GUI. The application retrieves data from a server in JSON format. The JSON serves both as a data source for offers and as a manifest containing URLs to images, which are downloaded subsequently. The app includes a caching mechanism, enabling offline operation after the initial launch.
